# Blue Heron Parents

A React Native app built with Expo.

## Getting Started

### Installation

```bash
npm install
```

### Running the App

#### Start the development server
```bash
npm start
```

#### Run on iOS Simulator (default)
```bash
npm run ios
```

#### Run on iPhone 16 Simulator
```bash
npm run ios:iphone16
```

#### Run on Android
```bash
npm run android
```

#### Run on Web
```bash
npm run web
```

## Available Scripts

- `npm start` - Start the Expo development server
- `npm run ios` - Launch the app in the default iOS simulator
- `npm run ios:iphone16` - Launch the app in the iPhone 16 simulator
- `npm run android` - Launch the app on an Android emulator or device
- `npm run web` - Launch the app in a web browser
